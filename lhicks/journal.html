17 February

Today I learned more about the application and usefulness of arrays. They are a collection of data points called elements. I used them to make a JavaScript program for changing the background color of and HTML page to a random color within the RGB color range. To do this, I used the random generation process:

  1. Generate an array of 3 random elements
  2. Convert the array into RGB format
  3. Output the RGB color to the background of the HTML page

3 March

Today the focus was on array methods, namely push(), pop(), shift(), and unshift(). The methods perform the following actions:

  • shift(): erases array elements from the end
  • unshift(): adds array elements to the end
  • pop(): erases elements from the beginning
  • push(): adds elements to beginning of array

"Weathery" Weather App

"Weathery" Weather App

Weather app created using Forecast.io with GPS Intergeneration.

Weathery is a weather app created using Forecast.io to fetch weather data from a latitude and longitude location and compile it into JSON Data. Although it isn't familiar with City locations or Zip Codes, android built in GPS is able to find the latitude and longitude, along with city name and direct address, zip code and etc.

Using this information, we are able to change the text fields on the MainActivity to refresh the current weather data for that time. A special new feature includes, color schemes, the current temperature affects the color of the screen.

Download Weathery @GooglePlay

Can You Even Puzzle Game App

Can You Even Puzzle Game App

A Math Puzzle Game, designed using simple Java, random integers and addition.

Can You Even is designed using Java, which calls two random numbers, and determines if they are odd or even. It is up to the user to determine if the numbers added together are odd or even.



Download Can You Even @GooglePlay
